# Ecomerace RentCar Backend

Express.js backend for the Ecomerace RentCar application with MongoDB, JWT authentication, Google OAuth 2.0, and email verification.

## Setup

1. Install dependencies:
```bash
npm install
```
```


3. Start the server:
```bash
npm run dev
```
